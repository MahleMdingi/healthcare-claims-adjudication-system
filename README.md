# Healthcare Claims Adjudication System

## Overview

This project is a SQL Server database design and backend development project that simulates a healthcare claims adjudication system used within the medical aid and health insurance industry.

The purpose of the system is to automate healthcare claims processing by validating claims against business rules, benefit structures, waiting periods, and medical plan coverage.

---

# Business Problem

Medical aid organizations process large volumes of healthcare claims daily.

Manual claims processing can:

* consume significant operational resources
* slow down claims turnaround times
* increase operational costs
* increase the risk of human error

This project aims to simulate a centralized claims adjudication system capable of automating core healthcare insurance validation workflows.
